O nome das pastas neste diretório corresponde ao nome das páginas significativas de dado site.

Dentro dessas pastas:

    Os arquivos "*-min.css" usam como referência os arquivos "*.css", que, por sua vez, são resultados da compilação dos arquivos "*.scss".

    Em meus projetos, uso a versão minificada do CSS na produção, a versão do SCSS para desenvolvimento, e a versão não minificada do CSS mormente em casos em que precise depurar o Sass.
