Este projeto foi feito usando Koala (http://koala-app.com), uma GUI que provê acesso a ferramentas diversas para o desenvolvimento de projetos no front-end.

Dentre os recursos do Koala, foram usados o Sass 3.5, Clean CSS 4.1, Autoprefixer 7.1 e Uglify JS 3.1

Por fim, para a avaliação desse projeto, é assumido que se tenha acesso a um servidor que use uma versão do PHP igual ou superior a 7.
