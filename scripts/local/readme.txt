O nome das pastas neste diretório corresponde ao nome das páginas significativas de dado site.

Dentro dessas pastas:

    Os arquivos "*-min.js" são para produção, e "_*.js" para desenvolvimento.

    "*-min.js" toma como referência o "*.js" do mesmo diretório, que, por sua vez, unifica os módulos do "_*.js".

    Livrarias são, contudo, adicionadas à versão de produção da página em arquivos separados.
