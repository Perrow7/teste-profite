theWheel é uma livraria de JavaScript que desenvolvo, sendo usada por padrão para meus projetos da web.

No momento, está passando por refinamentos para deixar algumas de suas funções mais estáveis, legíveis, sucintas [e/ou] rápidas; a seção de strings, particularmente, ainda será completamente reformulada para que incorpore expressões regulares em seus métodos.

Ainda assim, ela como um todo já pode ser usada como parâmetro adicional para se estimar minhas habilidades com o JavaScript, se isso for de interesse.

O arquivo "theWheel", referência para o arquivo "theWheel-min", contém apenas as funções da livraria usadas neste projeto; o arquivo "_theWheel" contém todas as funções da livraria até então feitas.
